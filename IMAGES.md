# Image Manifest

Drop photos into the `images/` folder using the **exact filenames** below. The site auto-detects `.jpg`, `.jpeg`, `.png`, and `.webp` — use whichever format makes sense. Recommended format is JPG at 80–85% quality, WebP if you need smaller file size.

If a file is missing, the placeholder stays visible — nothing breaks. Add photos one at a time as you have them; refresh the page between drops.

## Image best practices for this site

- **Mood** — Cinematic, low-key. Avoid bright/flat marketing photography. The site is dark violet — high-contrast natural light reads best.
- **Colour grading** — Slight warm push helps the gold accents sing. Cool / desaturated also works against the violet bg.
- **No logos or watermarks** — the design carries its own marks.
- **Avoid stock photography** — every image should feel like it's actually MCTC, not anonymous.
- **Compression** — Target under 250KB per image where possible. Use WebP for the big hero (`faculty_group_photo`) if file size matters.
